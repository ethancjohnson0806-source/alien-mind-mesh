# Alien Mind Mesh Network

This project implements a multi-node mesh network featuring the "Alien Mind" autonomous AI field.

## Project Structure

- `alien_mind_v10.py`: The core autonomous AI field implementation (v10.0).
- `ghost_mesh.py`: UDP-based network layer for the DHT (Distributed Hash Table).
- `ghost_ai_node_v10.py`: Bridge connecting the AI mind to the mesh network.
- `v10_softened_mesh.py`: Orchestration script for a mesh with a "softened" mind state.
- `threadball_injection.py`: Utility to inject text content into the mind's semantic field.

## How it Works

The system uses a DHT to distribute data across multiple nodes. Each node runs an instance of the Alien Mind, which can generate responses, store them on the mesh, and "learn" from transcripts generated by other nodes.

## Running the Demo

To run the v10 softened mesh orchestration:
```bash
python3 v10_softened_mesh.py
```

This will spawn multiple nodes, demonstrate distributed storage, and show the AI nodes interacting and learning from each other.
